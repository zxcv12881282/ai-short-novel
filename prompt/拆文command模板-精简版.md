# 📚 短篇小说拆文分析Command（精简版）

## 🎯 拆文分析框架

### 一、基础信息（必填）
```
故事名称：[标题]
全文字数：[约XXK]
题材类型：[打脸爽文/复仇爽文/重生爽文等]
核心冲突：[一句话概括]
```

### 二、故事梗概（200字内）
用一段话讲清楚：主角是谁→遇到什么问题→如何解决→最终结局

### 三、故事结构（起承转合）
**起**：开篇冲突（2-3个要点）
**承**：矛盾升级（2-3个要点）
**转**：反转揭露（2-3个要点）
**合**：结局收尾（2-3个要点）

### 四、核心人物
**主角**：设定/性格/目标/成长弧光（各1句话）
**反派**：设定/动机/手段/结局（各1句话）
**关键配角**：作用（1句话）

### 五、微创新点（3-5个）
与同类题材相比的新意：
- 创新点1
- 创新点2
- 创新点3

### 六、冲突与爽点（核心）
**冲突升级**（3-5层即可）：
- 第1层：[发生了什么] → 为什么让人气愤
- 第2层：[发生了什么] → 为什么让人气愤
- 第3层：[发生了什么] → 为什么爽

**核心爽点**（3-5个）：
- 爽点1：[具体表现] + 为什么爽 ★★★★☆
- 爽点2：[具体表现] + 为什么爽 ★★★★★

### 七、痛点共鸣（3-5个）
- 痛点1：[名称] - 为什么能引发共鸣
- 痛点2：[名称] - 为什么能引发共鸣

### 八、写作技巧（挑重点）
只写最突出的3-5个技巧：
- 技巧1：[具体表现]
- 技巧2：[具体表现]

### 九、衍生选题（2-3个）
**选题1：《标题》**
梗概（50字）+ 核心冲突

**选题2：《标题》**
梗概（50字）+ 核心冲突

### 十、总结
**爆款密码**：3-5个核心要素
**推荐学习点**：3个
**可改进点**：2个（选填）

---

## 📏 精简原则

1. **控制篇幅**：拆文长度为原文的1/3左右
2. **提炼精华**：只写最重要的点，不展开过多细节
3. **简明扼要**：每个点用1-2句话说清楚
4. **删繁就简**：合并相似内容，避免重复
