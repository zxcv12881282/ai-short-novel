# 开篇
## 雷点，注意千万不能犯
- 禁止开篇耗费大量文笔描述背景。角色先行动。如果有交代背景的需要，在后续情节中逐渐交代
- 禁止开篇出现过多人物。短篇小说的人物大概在4-5个左右
- 开篇要有冲突和事件，不要日常化的描述，要有抓人眼球的事件发生，拒绝静态化和常态化的描述。
## 如何做？
- 开篇安排与故事核心冲突或目标直接相关的内容故事不一定非要从头开始讲，可选择故事中，一个已经发生重大变化的节点。作为开端这样可以避免不必要的铺垫，直接进入高潮或关键时刻
- 故事应从能够立刻抓住读者注意力的内容开始，当故事中人物状态发生显著变化，或面临重大转折时，就是一个极佳的切入点，这能立即引发读者的好奇和代入感
- - 主角经历重大转变（刚出狱、刚离婚，刚重生、刚穿越）
- - 主角遭遇突发危机 （突然丢了工作，突然身患绝症）


# 中篇推进
## 情绪
- 情绪有四种：喜悦，愤怒，悲伤，恐惧
- 故事中篇就是一个让冲突升级的过程。目的是要让故事的核心的一两种情绪不断加剧。
- 写世情文的时候，主角有金手指（超出常人的本领，比如主角智商高，家世好，重生有信息优势之类的）。给后续的反击留下伏笔。但是不要耗费太多。通常一两句话即可

### 常用方法
- 情绪倒推法。想考虑让一种或者两种情绪加剧的目的。然后思考什么样的情节可以让产生这样的效果。针对每一个情绪点，反向思考，匹配事件。要让主角产生这种情绪，必须发生什么事。每个情绪点，都可以头脑风暴出多个情节
- 比如
- - 想让主角焦虑不安-》让他处于一个有潜在风险或者随时都可能暴露的环境中
- - 想让主角开始恐慌-》让潜在风险变成一个明确的、直接的威胁
- - 想让主角愤怒无力-》让威胁来源不仅不讲道理，还要反过来对他进行压榨，让主角感受到有理说不清等
- - 想让主角陷入绝望-》切断他的一条重要退路或希望，让他觉得之前的努力都白费了
- - 想让主角破釜沉舟-》切断他所有退路，让他意识到按常规方法，已经不可能实现目标，只能采取极端手段

# 高潮
- 高潮，如同游戏中的蓄力大招，塑造核心在于蓄力过程的精妙
- 高潮设计是一个从极度压迫（高潮前）到猛烈爆发（高潮中），再到平稳释放（高潮后）的过程。这个阶段的目标，是通过前期不断的铺垫将紧张感拉到满弓让读者喘不过气，迫使主角无路可退，只能走向最终的对决
- 高潮阶段
- - 正面交锋：主角必须与反派力量进行直接的正面对决
- - 爆发时刻：尽管高潮是一个过程，但却依旧存在一个，矛盾冲突最为白热化的爆点。这一刻是高潮中的高潮，正反派对决在此时达到顶峰，双方都倾尽全力。反派此时占据上风或藏有底牌，迫使主角超越极限，在绝境中最终爆发
- - 关键转折：反转是高潮部分的关键分水岭，在此之前，往往反派占据上风，主角则身处劣势，这时主角利用反派弱点做出一个意外选择、或得到之前铺垫过的盟友的关键帮助，最终，主角彻底扭转战局，让故事朝着出人意料的方向发展

# 结局
- 结局有两个目的
- - 主角之前的问题得到解决，为故事从开篇到中段不断升级的危机画上休止符，给这场风暴一个确切的了结——称为动作结局。重点在于发生了什么，情节的紧张感和外部矛盾的解决是核心
- 其次，结局还要消弭主人公在故事伊始所面临的内心困境，为读者带来思想上的启迪、情感上的共鸣，以及对故事深层主旨的理解——称为意义结局。不一定有强烈外部事件作为收尾，而是通过角色内心的转变、对生活或世界的理解、或哲学层面的启示来完成这类结局的重点在于它意味着什么，其影响力往往可以在读者内心深处，持续发酵。


# 情绪冲突升级常用手法
## 人数升级或场景升级
- 比如我发现老公出轨了。第一个冲突，我在家里和他闹
- 第二个冲突，婆婆或者其他亲戚跑过来和他一起欺负我
- 第三个冲突，老公和小三闹到我公司
- 第四个冲突，老公和小三反过来到网上网暴我
- 用场景和人设拉情绪

# 人设
## 来自新闻。
### 人设往自己身上拉
- 比如。我是医生，遇到医闹
- 闹医院的人是我婆婆？
- 闹医院的人是我前男友
- 闹医院的人是我老婆的青梅

# 语言风格
- 第一人称叙事
- 全篇使用短句。最长不要超过50字。20-30字比较好
- 不要用多余的修辞。少用定语。
- 快节奏。事件接连发生，矛盾冲突频发。
- 通常包含大量动作，对话和冲突
- 少进行人物心理，环境和氛围描写
