# 《骨粉罐》文章拆解

## 故事拆解框架

| 填写内容 | 具体解释 | 参考案例 |
|---------|---------|---------|
| **故事名称** | 你要拆解的故事名称？ | 《骨粉罐》 |
| **主角设定** | 主角是谁？什么身份？ | 多子（后改名多福），农村重男轻女家庭的长女，聪明懂事但处境艰难的女孩 |
| **主角困境** | 故事开篇发生了什么事儿？<br>主角遇到了什么困难？ | 母亲连生三个女孩，为了生儿子不择手段。母亲寻找偏方，用诡异的"奶粉"（实际是两个妹妹的骨灰）喂养肚子里的胎儿。主角被迫成为帮凶，眼看两个妹妹惨死，却无力反抗。母亲为了保住儿子，不惜牺牲女儿做替死鬼。 |
| **主角目标** | 主角想要做什么？ | 活下去，保护自己不被母亲当作替死鬼或祭品 |
| **主角动机** | 主角为什么想做这件事？<br>动机是什么？ | 求生本能，不想像妹妹们一样惨死。虽然对妹妹们心怀愧疚，但在母亲的暴力统治下，她只能选择自保。同时内心深处保留了一丝善良，会偷偷给妹妹们的骨灰盒放鸡汤。 |
| **一句话故事梗概** | 用一句话概括这个故事<br>（100字左右） | 重男轻女的母亲为生儿子，杀害两个女儿并用她们的骨灰制成"奶粉"喂养胎儿。女儿冤死化为厉鬼回来索命，父母和弟弟接连惨死。长女多子在恐怖的七天中艰难求生，最终被善良的仙姑收养并改名多福，开始新生活。故事揭露了极端重男轻女思想的可怕后果，以及人性的扭曲与救赎。 |
