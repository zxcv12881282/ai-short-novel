# 《心安则灵》文章拆解

## 故事拆解框架

| 填写内容 | 具体解释 | 参考案例 |
|---------|---------|---------|
| **故事名称** | 你要拆解的故事名称？ | 《心安则灵》 |
| **主角设定** | 主角是谁？什么身份？ | 唐宇安，实习记者，贫困家庭出身，为了给母亲治病而被迫冒充哥哥周辰的身份 |
| **主角困境** | 故事开篇发生了什么事儿？<br>主角遇到了什么困难？ | 五年前母亲因病去世欠下巨债，同父异母的哥哥周辰给他50万，让他冒充自己的身份照顾失明的未婚妻许灵心两年。两年后许灵心复明，周辰重新出现，唐宇安被迫离开。三年后他毕业回国做记者，却在工作中不断遇到许灵心和周辰，还被周辰陷害污蔑。 |
| **主角目标** | 主角想要做什么？ | 为二十年前被周家陷害的母亲平反昭雪，扳倒周家，揭露周辰的真面目 |
| **主角动机** | 主角为什么想做这件事？<br>动机是什么？ | 母亲当年被周家大小姐设局陷害，背负"暗娼"污名郁郁而终。周辰为了享受豪门生活，亲手把母亲推入深渊，甚至连墓地钱都不肯出。唐宇安要为母亲讨回公道，让害死母亲的人付出代价。 |
| **一句话故事梗概** | 用一句话概括这个故事<br>（100字左右） | 唐宇安为了50万冒充哥哥周辰照顾失明未婚妻许灵心两年，两人相爱却因身份不敢相认。三年后重逢，许灵心早已识破真相，一直在等他回来。两人联手扳倒周辰和周家，为唐宇安被冤枉的母亲平反昭雪。故事讲述了身份错位的爱情、为母复仇的决心，以及两个在困境中相互救赎的灵魂终成眷属的故事。 |
