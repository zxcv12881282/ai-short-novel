# 《不死的蒲公英》文章拆解

## 故事拆解框架

| 填写内容 | 具体解释 | 参考案例 |
|---------|---------|---------|
| **故事名称** | 你要拆解的故事名称？ | 《不死的蒲公英》 |
| **主角设定** | 主角是谁？什么身份？ | 郑夏夏，农村家庭长女，天资平庸但异常努力的普通女孩 |
| **主角困境** | 故事开篇发生了什么事儿？<br>主角遇到了什么困难？ | 家里没有儿子，在重男轻女的农村环境中，父母被歧视欺负，分家只分到破土坯房。妹妹出生后更聪明，父母将所有资源倾斜给妹妹。夏夏中考失利只能上中专，面试外企时因学历被拒之门外，人生陷入困境。 |
| **主角目标** | 主角想要做什么？ | 考上大学改变命运，让父母在村里抬起头，证明女儿不比儿子差，摆脱被发卖打工或早婚的命运 |
| **主角动机** | 主角为什么想做这件事？<br>动机是什么？ | 心疼父母因没儿子被欺负歧视，不想重复村里女孩早婚早育、沦为生育工具的悲惨人生，想要掌握自己的命运，为家人争气，证明女性的价值 |
| **一句话故事梗概** | 用一句话概括这个故事<br>（100字左右） | 农村女孩郑夏夏天资平庸却异常努力，在重男轻女的环境中艰难成长。中考失利读中专后因学历被外企拒绝，她决心考大学。历经被逼婚、资源倾斜妹妹、父亲中风等挫折，她咬牙坚持，最终考上二本，又克服家庭阻碍两次考研成功上岸985名校，找到年薪30万的工作，嫁给博士男友，用坚韧和努力如蒲公英般顽强生长，改变了自己和家人的命运。 |
